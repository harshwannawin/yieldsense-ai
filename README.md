# EcoCart - Smart Grocery AI

Dynamic pricing grocery app powered by Groq AI.

## Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Create a `.env` file in the root directory:
   ```
   GROQ_API_KEY=your_groq_api_key_here
   ```

4. Run the app:
   ```bash
   streamlit run app.py
   ```

## Features

- Real-time dynamic pricing using AI
- Seller dashboard for inventory management
- Customer storefront with personalized pricing
- AI-powered discount strategies based on user persona

## Tech Stack

- Streamlit
- Groq AI (Llama 3.3)
- Python
